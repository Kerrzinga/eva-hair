# Eva Hair Static Site

Clean static rebuild based on publicly available content from `https://evahair.co.nz/`.

Open `index.html` to browse locally. The site uses only local CSS, JavaScript and image assets; outbound map, social and LA BIOSTHETIQUE links remain external by intent.

The contact form is currently display-only because the WordPress backend/account is not available. It must be connected to the client's chosen form or booking service before publishing.
